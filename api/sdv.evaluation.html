sdv.evaluation module¶

Tools to evaluate the synthesized data.

Functions

evaluate(synth[, real, metadata, root_path, …])

Compute a score using SDMetrics.
sdv.evaluation.evaluate(synth, real=None, metadata=None, root_path=None, table_name=None, get_report=False)[source]¶

Compute a score using SDMetrics.

Parameters

  • synth (dict[str, pandas.DataFrame] or pandas.DataFrame) – Map of names and tables of synthesized data.

  • real (dict[str, pandas.DataFrame] or pandas.DataFrame) – Map of names and tables of real data.

  • metadata (str, dict, Metadata or None) – Metadata instance or details needed to build it.

  • root_path (str) – Relative path to find the metadata.json file when needed.

  • descriptors (list[callable]) – List of descriptors.

  • metrics (list[callable]) – List of metrics.

  • table_name (str) – Table name to be evaluated, only used when synth is a pandas.DataFrame and real is None.

  • get_report (bool) – whether to return the complete SDMetrics report. If False (default), only the overall score is returned.

Returns

float or sdmetrics.MetricsReport
