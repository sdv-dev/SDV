Modeler class.

The Modeler class applies the CPA algorithm recursively over all the tables from the dataset.

Parameters

  • metadata (Metadata) – Dataset Metadata.

  • model (type) – Class of model to use. Defaults to copulas.multivariate.GaussianMultivariate.

  • model_kwargs (dict) – Keyword arguments to pass to the model. Defaults to None.
